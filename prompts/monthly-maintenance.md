# Monthly Maintenance Prompt

Help me refresh this Engagement Stack.

Read:

- `maintenance/monthly-review.md`
- `setup/completeness-audit.md`
- `AGENTS.md`

Ask what changed this month:

- new work shipped
- new repos or live products
- new metrics
- changed engagement preferences
- stale links
- resume updates needed
- new target roles or clients

Then update Markdown and YAML together.
