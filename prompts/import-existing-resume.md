# Import Existing Resume Or CV

Use this when the user provides an existing resume, CV, LinkedIn export, website bio, or work-history document.

Steps:

1. Extract canonical facts into `cv/`.
2. Ask what has changed since the document was written.
3. Ask which claims have proof links.
4. Ask current engagement preferences.
5. Generate missing YAML sidecars.
6. Produce a gap list: missing dates, unclear roles, unsupported claims, stale links.

Do not overwrite canonical facts without confirmation.
