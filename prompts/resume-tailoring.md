# Resume Tailoring Prompt

I want to tailor my resume from this Engagement Stack for a specific role.

Read:

- `AGENTS.md`
- `cv/canonical-cv.md`
- `cv/work-history.md`
- `cv/skills.md`
- `portfolio/README.md`
- relevant `portfolio/projects/*`
- `resume-shaders/README.md`
- `evidence/README.md`

Then read the target role description I provide.

Return:

1. a fit summary;
2. the strongest evidence-backed claims to foreground;
3. claims to avoid because evidence is weak;
4. a proposed resume shader, or an update to an existing shader;
5. a tailored resume source file generated through that shader;
6. an ATS field pack if useful.

Do not alter canonical facts unless I confirm an update.
