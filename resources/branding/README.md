# Brand Assets

Paired image assets for Engagement Stack's GitHub and social presence.

| Asset | Size | Path | Use |
| --- | --- | --- | --- |
| README banner | 1500x500 SVG / 1280x427 PNG | `banner/engagement-stack-banner.svg` / `banner/engagement-stack-banner.png` | Top of `README.md` |
| Social preview | 1200x627 SVG / PNG | `social/engagement-stack-linkedin-card.svg` / `social/engagement-stack-social.png` | GitHub repo social preview, LinkedIn/X embeds |

## Regenerate

```sh
python3 resources/branding/generate.py
```

The generated images intentionally match the `job-search-superpower` family: dark navy gradient, warm gold accent, left title bar, subtle geometric marks, and pipe-separated capability text. Gemba informs the README conventions; Job Search Superpower informs the visual family.
