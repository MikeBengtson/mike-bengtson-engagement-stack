# Cover Letter

A one-page cover letter rendered with the navy theme (`styles/navy.css`, letter
density) so it matches your resume as a set.

- [`cover-letter.md`](cover-letter.md) is a template — replace the `[BRACKET]` placeholders.
- Rewrite the **"why this company"** paragraph for every application; it is the part
  hiring managers read most closely.
- Keep claims evidence-backed (cite `cv/` and `portfolio/`); do not invent.

Render:

    scripts/render-resumes.sh cover-letter   # -> generated/resumes/cover-letter.pdf
