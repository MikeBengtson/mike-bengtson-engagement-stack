# Framework Origin

Engagement Stack is a public-first framework for maintaining a human-readable, agent-readable professional dossier.

Canonical framework repository: `https://github.com/MikeBengtson/engagement-stack`

## Inspiration

Engagement Stack is inspired in part by Every Inc.'s `after-automation-agent-mode` project:

https://github.com/EveryInc/after-automation-agent-mode

The borrowed idea is structural, not domain-specific: a repository can act as an agent working surface by combining local instructions, source materials, prompts, review criteria, and generated outputs in one navigable project. Engagement Stack adapts that pattern for professional identity, CVs, portfolios, engagement preferences, and job-search materials.

## Category

Engagement Stack treats a professional career as a maintained source-of-truth system:

- human-readable Markdown for context and narrative;
- agent-readable YAML for matching, scoring, routing, and generation;
- evidence-backed project claims;
- explicit engagement preferences;
- submission adapters for job platforms and direct engagement requests.
